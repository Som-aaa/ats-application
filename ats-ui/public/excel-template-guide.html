  • Column C (Job Description) must contain the actual job description text

  • Job descriptions should be detailed and contain relevant keywords

üîß How to Create This File

  1. Open Excel, Google Sheets, or any spreadsheet application
  2. Create 3 columns with headers: Company Name, Job Role/Title, Job Description
  3. Fill in your data starting from row 2 (row 1 can be headers)
  4. Save as .xlsx or .xls format
  5. Upload to the ATS system

‚ùå Common Mistakes to Avoid

  • Putting job descriptions in Column A - They must be in Column C
  • Empty job descriptions - Column C cannot be empty
  • Wrong file format - Use .xlsx or .xls, not .csv or .txt
  • Missing data - Ensure you have at least one row of data

‚úÖ Troubleshooting

If you get the error "No job descriptions found in the Excel file":

  1. Check that your job descriptions are in Column C (third column)
  2. Ensure Column C is not empty
  3. Verify you have data beyond the header row
  4. Make sure the file is saved in .xlsx or .xls format
