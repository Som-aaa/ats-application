How to format your job descriptions Excel file for bulk analysis

üìã File Format Requirements

Your Excel file should follow this 3-column structure for optimal resume renaming:

‚úÖ Correct Format:

  • File type: .xlsx or .xls
  • Column A: Company Name
  • Column B: Job Role/Title
  • Column C: Job Description
  • Skip the first row (it can be a header like "CompanyName", "Role", "Description")
  • Maximum file size: 5MB

üìù Example Structure

Here's how your Excel file should look:

Row 1: CompanyName | Role | Description (header - optional) Row 2: Markforged | Software Engineer II | The text describes Markforged as a company leading the digital transformation... Row 3: Microsoft | Data Scientist | We need a specialist in machine learning, Python programming... Row 4: Apple | Product Manager | We are seeking a product leader with 3+ years experience...

üîë Why This Format?

üí° Benefits of the 3-column format:

  • Accurate Resume Renaming: Company and role names are extracted directly from your Excel columns
  • No AI Extraction Errors: Eliminates the need for AI to guess company/role from description text
  • Personalized Naming: Resumes will be renamed as: Username_CompanyName_RoleName_ID
  • Better Organization: Clear separation of company, role, and description information

‚ö†Ô∏è Important Notes

üö´ What NOT to do:

  • Don't put job descriptions in other columns (D, E, F, etc.)
  • Don't merge cells
  • Don't use complex formatting or formulas
  • Don't include images or charts
  • Don't put company/role info in the description column

üí° Pro Tips:

  • Company Names: Use consistent naming (e.g., "Google Inc" not "Google, Inc." or "Google")
  • Role Titles: Be specific but concise (e.g., "Senior Software Engineer" not "Software Engineer II")
  • Descriptions: Keep detailed but focused on requirements and responsibilities
  • Test First: Try with a small file (2-3 job descriptions) before bulk processing

üîß How to Create the File

Step-by-step instructions:

  1. Open Excel, Google Sheets, or any spreadsheet application
  2. Column A: Start typing company names
  3. Column B: Add corresponding job roles/titles
  4. Column C: Write detailed job descriptions
  5. Each job posting goes in a separate row
  6. Save the file as .xlsx or .xls format
  7. Upload to our system and analyze!
